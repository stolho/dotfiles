# My dotfiles
Various dotfiles:
* bash_profile
* bashrc
* tmux_conf
* vimrc

## Installation

### dotfiles
```bash
$ ./install.sh
```

### Python3
#### macOS
```bash
$ brew install python3
```

### Virtualenvwrapper

#### macOS
To install virtualenvwrapper on macOS run:
```bash
$ pip3 install virtualenvwrapper
```

### Golang
#### macOS
```bash
$ brew install golang
```
